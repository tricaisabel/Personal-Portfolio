About me

I started my programming journey with C++ in high-school and then got more in-depth with algorithms, data structures and object oriented programming. I later started tutoring OOP in C++.

Then, I discovered web development, took online programming courses (freeCodeCamp, the Odin Project, Youtube Tutorials, etc.) and then I started building websites on my own using my previous the graphic design skills, a hobby I had for a very long time.

Last year I started working on my bachelor's degree project and I'm excited to apply all the new things I learned. I am building a project management website using the MERN stack that helps teams to organise according to the Agile methodology.

I was memeber of the student's syndicate and had previous part-time working experience in graphic design, but I wanted to postpone getting a full time programming position after I graduated because I felt I didn't have as much time as I wanted to dedicate. I am committed to all of my projects and as a result managed to get several performance scholarships in the last 3 years.
